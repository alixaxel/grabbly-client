What
====

This is a remake of excellent http://wordsquared.com/ game, client side.

What is copied from Wordsquared: board-bg-fill.png, buttons.png, tiles.png. These will be replaced
by custom graphics if Wordsquared is against reuse or when better graphics will be found.

If anyone from Wordsquared team has any questions or issues with this project, please,
contact me at temotor@gmail.com .

Rest assured, there are absolutely no scam or fishing intentions in this project.


Features
========

- Speed.
- No minimap. Anti-feature? WTF! But really, is there any point in minimap on Wordsquared?
- Russian wordlist. Sadly enough, it was my primary concern.
